数组：不能更改长度，值类型
语法：
```
var 数组名 [数组长度] 数据类型
```
var nums [5] int
var nums = [...] int{10,20,30,40,50}  //自动确定数组长度

遍历数组
```
for i := 0;i < len(a);i++{
    fmt.Println(a[i])
}
```
for _,value := range a{         //_匿名变量
    fmt.Println(value)
}

[2][3]int:两个长度为3的数组

切片：变长数组，每个元素类型相同，引用类型
var 切片名 []类型
slice1 ：= make([]type类型,len长度,cap容量) //在内存中分配空间
s := [] int {1，2，3}
通过数组截取来初始化切片
```
a := [5] int {1,2,3,4,5}
s := a[:]   //截取数组a的所有元素
e.g.  s := a[1:3]  //[1,3)左闭右开：[1,2]
```
