error接口
type Error interface {
    Error() string
}
error通过返回值来表示错误，通常是返回值的最后一个参数
自定义error:实现error方法
errorf:返回错误类型并输出具体参数

defer:延迟一个函数或方法的执行，只能在方法函数内部使用
特性：
多个defer语句按先进后出（LIFO）的顺序执行

panic:
特性：
1. 程序崩溃
2. 可以接收任意类型的参数
3. 可以在defer语句中捕获

recover:
特性：
1. 只能在defer语句中使用
2. 必须在defer语句中调用
3. 只能捕获panic的参数
