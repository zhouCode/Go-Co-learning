-递归函数
func factorial(n int) int {
    //退出条件！！！
    if n == 0 {
        return 1
    }
    //递归调用
    return n * factorial(n-1)
}
- 指针
- 指针定义

- 指针赋值
var a int = 10

- 指针解引用
var a int = 10
var p *int = &a//p指向a的地址
fmt.Println(*p)//输出a的值
- 指针作为参数
func changeValue(p *int) {
    *p = 100//通过指针修改指向变量的值
}
- 指针作为返回值
func getPointer() *int {
    var a int = 10
    return &a//返回a的地址
}
- 1,对于一个变量，想获取他的地址，就使用：&变量名
- 2,对于一个指针变量，想获取他指向的变量，就使用：*指针变量名
- 3,声明一个指针变量，使用：*类型
